---
title: 'Update Sequence'
api: 'PUT /sequences/:id'
description: 'Update all fields of a sequence (full object replacement)'
---

## Path Parameters

<ParamField path="id" type="integer" required>
  Sequence ID
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Sequence name (max 200 characters)
</ParamField>

<ParamField body="description" type="string">
  Sequence description
</ParamField>

<ParamField body="instructions" type="string" required>
  AI instructions playbook
</ParamField>

<ParamField body="email_templates" type="array" required>
  Email templates array
</ParamField>

<RequestExample>
```json
{
  "name": "Outbound – Cold V2",
  "description": "Updated outreach sequence",
  "instructions": "Updated playbook with new approach...",
  "email_templates": [
    {
      "name": "Step 1 - Intro",
      "subject": "New subject",
      "body": "<p>Updated intro...</p>"
    }
  ]
}
```
</RequestExample>

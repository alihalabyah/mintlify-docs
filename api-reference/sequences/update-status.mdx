---
title: 'Update Sequence Status'
api: 'PATCH /sequences/:id'
description: 'Update only the status of a sequence (activate/pause/archive)'
---

## Path Parameters

<ParamField path="id" type="integer" required>
  Sequence ID
</ParamField>

## Body Parameters

<ParamField body="status" type="string" required>
  New status: `draft`, `active`, `paused`, `archived`
</ParamField>

<RequestExample>
```json
{
  "status": "active"
}
```
</RequestExample>
